# Spaxe

> Interactive 3D navigation for complex assemblies.

Spaxe lets you import STL files, manually build exploded views, inspect connected parts, and embed your assemblies anywhere. Built for makers and industrial teams.

🔗 **Live demo** → [spaxe.vercel.app](https://spaxe.vercel.app)

---

## Features

- 📦 **STL Import** — drag & drop, multi-part assemblies
- 🔩 **Exploded View** — manual XYZ positioning per part with transform handles
- 🔗 **Neighborhood** — visualize connected parts instantly
- 🧅 **X-Ray / Layer Peeling** — inspect inner components
- 🔗 **URL & PDF Links** — attach docs to any part
- 📤 **Iframe Export** — embed anywhere

## Stack

- **Framework** — Next.js 16 · TypeScript
- **UI** — Tailwind CSS · shadcn/ui
- **3D** — Three.js · React Three Fiber · React Three Drei · three-stdlib · React Three Postprocessing
- **State** — Zustand
- **Hosting** — Vercel

## Getting Started

```bash
git clone https://github.com/MatthieuLOBJOIS/spaxe.git
cd spaxe
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
